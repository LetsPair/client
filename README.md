# Let's Pair

## Setting up

1. Grab the repo
```bash
git clone https://github.com/LetsPair/client.git
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Run the server
```bash
npm run dev
# or
yarn run dev
```

### Some things you'll need

#### Environment Variables

| Name                      | Purpose |
|---------------------------|---------|
| `PORT`                    | This is the port to run the application at |
